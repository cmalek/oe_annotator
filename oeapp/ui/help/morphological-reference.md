# Morphological Tag Reference

Quick reference guide for Old English morphological tags used in annotations.

## Cases

| Code | Name | Description |
|------|------|-------------|
| `n` | Nominative | Subject case |
| `a` | Accusative | Direct object case |
| `g` | Genitive | Possessive case |
| `d` | Dative | Indirect object case |
| `i` | Instrumental | Instrument/manner case |

## Numbers

| Code | Name |
|------|------|
| `s` | Singular |
| `p` | Plural |

## Genders

| Code | Name |
|------|------|

| `f` | Feminine |

## Verb Classes

### Weak Verbs

| Code | Name | Description |
|------|------|
| `w1` | Weak Class I | Verbs like `nerian` (to save) |
| `w2` | Weak Class II | Verbs like `lufian` (to love) |
| `w3` | Weak Class III | Verbs like `habban` (to have) |

### Strong Verbs

| Code | Name | Description |
|------|------|
| `s1` | Strong Class I | Verbs like `rīdan` (to ride) |
| `s2` | Strong Class II | Verbs like `cēosan` (to choose) |
| `s3` | Strong Class III | Verbs like `drincan` (to drink) |
| `s4` | Strong Class IV | Verbs like `beran` (to bear) |
| `s5` | Strong Class V | Verbs like `sprecan` (to speak) |
| `s6` | Strong Class VI | Verbs like `faran` (to go) |
| `s7` | Strong Class VII | Verbs like `hātan` (to be called) |

## Common Abbreviations

These abbreviations appear in exports:

- `pron:rel` - Relative pronoun
- `pron:pers` - Personal pronoun
- `pron:dem` - Demonstrative pronoun
- `nom` - Nominative
- `acc` - Accusative
- `dat` - Dative
- `gen` - Genitive
- `sg` - Singular
- `pl` - Plural
- `m` - Masculine
- `f` - Feminine
- `n` - Neuter

## Compact Export Format

In DOCX exports, annotations use compact formats:

- `dat1` = Dative singular
- `acc1` = Accusative singular
- `nom.pl` = Nominative plural
- `w2` = Weak Class II
- `s3` = Strong Class III
