# Keyboard Shortcuts

## Navigation

| Action | Shortcut |
|--------|----------|
| Next Sentence | **J** |
| Previous Sentence | **K** |
| Next Token | **→** (Right Arrow) |
| Previous Token | **←** (Left Arrow) |
| Focus Translation Field | **T** |

## Annotation

| Action | Shortcut |
|--------|----------|
| Annotate Selected Token | **A** |
| In Annotation Modal: | |
| - Apply Annotation | **Enter** or **Return** |
| - Cancel | **Escape** |
| - Select POS (Noun) | **N** |
| - Select POS (Verb) | **V** |
| - Select POS (Adjective) | **A** |
| - Select POS (Pronoun) | **R** |
| - Select POS (Determiner) | **D** |
| - Select POS (Adverb) | **B** |
| - Select POS (Conjunction) | **C** |
| - Select POS (Preposition) | **E** |
| - Select POS (Interjection) | **I** |

## Editing

| Action | Shortcut |
|--------|----------|
| Save Project | **Ctrl+S** |
| Undo | **Ctrl+Z** |
| Redo | **Ctrl+Y** or **Ctrl+Shift+Z** |
| Export to DOCX | **Ctrl+E** |

## Help

| Action | Shortcut |
|--------|----------|
| Show Help | **F1** |

## Workflow Tips

1. Use **J** and **K** to quickly navigate between sentences
2. Use **→** and **←** to move between tokens within a sentence
3. Press **A** when a token is selected to open the annotation modal
4. In the annotation modal, use single-letter shortcuts (N, V, A, etc.) to quickly select Part of Speech
5. Press **Enter** to apply annotations and return to editing
6. Use **T** to quickly jump to the translation field for the current sentence
